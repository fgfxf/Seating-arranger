
# 随机排座系统

随机排座，支持导入名单和导出排座结果。

**特点**：

1. 可以ban掉某个位置

2. 可以手动拖拽

3. 可以固定某个人的座位

## 在线体验

   https://fgfxf.github.io/Seating-arranger/

## 本地使用

**要求:**  Node.js 24.2.0

1. 初始化项目
   `npm init -y`
   
2. 安装Vite
	`npm install vite --save-dev`
	
3. 构建
    ​`npx vite build`

4. target at dist/

    `python -m http.server`
